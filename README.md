# F1 Telemetry Project

Real-time telemetry system for F1-style racing data collection and visualization.

## System Architecture

- **Backend**: Python FastAPI (REST + WebSockets)
- **Frontend**: React + TypeScript (Vite)
- **Hardware**: ESP32 with sensors (GPS, IMU, Temperature)
